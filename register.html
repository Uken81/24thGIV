Join and Register on The 24th Georgia Discord

Make sure to visit and read ALL of the info in the Rules channel.

Make sure to download Teamspeak as it is our primary communication tool in game and then book mark this channel "the-irish-legion.teamspeak3.com"

You WILL need to change your in game name for events to avoid being kicked
You can either change your steam profile name or Watch This Video to learn how to apply name change only in War of Rights(recommended).

We suggest attending a drill before your first live event so we can give you a rundown of the basics and how we work together as a regiment
